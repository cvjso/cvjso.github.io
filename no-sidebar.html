Produto

O desafio lançado pelo Summer Job foi justamente melhorar a proposta de avaliação dos participantes, uma vez que o sistema atual, necessita de  uma ferramenta efetiva que otimize e transforme os dados recebidos dos mentores em informações úteis para o capital humano, para atender às demandas necessárias para uma possível contratação. 

Logo de acordo com nossas pesquisas e entrevistas, foi perceptível a escolha de um Web-App para fazer o processo de avaliação do Summer Job, assim como para visualizar os dados recolhidos do mentor, já que é um meio multi-plataformas, aonde é possivel o acesso à partir de todos os dispositivos.

Funcionamento

  Após acessar a página Home do website, o usuário se encontra com a página de login. Caso esse usuário não tenha um login, ele poderá se dirigir diretamente a página de cadastro. Mais adiante, ele se encontrará com o perfil dos participantes e seus co-workers, onde terá diferentes opções dependendo do seu relacionamento com a equipe.
Por exemplo, se for um Mentor, poderá escolher os critérios e avaliar os participantes da equipe através destes. Porém, se for alguém do Capital Humano, ele poderá visualizar todas as avaliações feitas pelos diferentes mentores, assim como comentários importantes que poderão ser feitos.
